About Our System

Learn how our sentiment-based product recommendation system works

Project Overview

This is a sentiment-based product recommendation system built for an e-commerce company 'Ebuss'. The system combines collaborative filtering with sentiment analysis to provide high-quality, personalized product recommendations.

Key Objectives:
  • Analyze product reviews and build a sentiment analysis model
  • Create both user-based and item-based recommendation systems
  • Combine sentiment analysis with collaborative filtering
  • Deploy the system with a user-friendly web interface
Machine Learning Models
Sentiment Analysis:
  • Logistic Regression
  • Random Forest
  • XGBoost
  • Naive Bayes
  • Neural Network Classifier
Recommendation Systems:
  • User-based Collaborative Filtering
  • Item-based Collaborative Filtering
  • Cosine Similarity
Text Processing:
  • TF-IDF Vectorization
  • Bag of Words
  • NLTK for preprocessing

Backend:

  • 20,000+ unique users

How It Works

1. Data Processing
  • Clean and preprocess review text
  • Handle missing values and outliers
  • Extract features using TF-IDF
  • Create user-item rating matrix
2. Model Training
  • Train multiple sentiment analysis models
  • Build collaborative filtering systems
  • Evaluate and select best models
  • Optimize hyperparameters
3. Recommendation Generation
  • Generate top 20 CF recommendations
  • Calculate sentiment scores for products
  • Combine CF and sentiment scores
  • Select top 5 final recommendations
4. Web Interface
  • User-friendly Flask web application
  • Real-time recommendation generation
  • Interactive results visualization
  • Responsive design for all devices
